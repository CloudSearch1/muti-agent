## 🎨 新 UI 设计规范

### 设计原则
1. **简洁高效** - 减少点击次数，核心功能一键可达
2. **实时反馈** - 所有操作都有即时视觉反馈
3. **渐进式披露** - 高级功能默认隐藏，按需显示
4. **一致性** - 统一的交互模式和视觉语言

### 色彩方案
- **主色：** 紫色系 (#667eea - #764ba2)
- **成功：** 绿色 (#10b981)
- **警告：** 黄色 (#f59e0b)
- **错误：** 红色 (#ef4444)
- **信息：** 蓝色 (#3b82f6)

### 布局结构
```
┌─────────────────────────────────────┐
│  顶部导航栏 (Logo + 主导航 + 用户)    │
├─────────────────────────────────────┤
│  侧边栏 (可选)  │  主内容区          │
│  - 快速导航      │  - 仪表盘/任务/... │
│  - 过滤器        │  - 数据可视化      │
│  - 最近访问      │  - 操作面板        │
├─────────────────────────────────────┤
│  底部状态栏 (系统状态 + 通知)         │
└─────────────────────────────────────┘
```
