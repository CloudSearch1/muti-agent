### 2. 后端 API 更新

#### GET /api/v1/tasks/{task_id}
```http
GET /api/v1/tasks/1
Response: {
  "id": 1,
  "title": "创建用户管理 API",
  "description": "...",
  "status": "in_progress",
  "statusText": "进行中",
  ...
}
```

#### PUT /api/v1/tasks/{task_id}
```http
PUT /api/v1/tasks/1
Body: {
  "status": "completed",
  "title": "新标题",
  "description": "新描述"
}
Response: {
  "status": "success",
  "message": "任务已更新",
  "task": { ... }
}
```

**支持的更新字段：**
- ✅ `status` - 任务状态
- ✅ `title` - 任务标题
- ✅ `description` - 任务描述
- ✅ `priority` - 优先级
- ✅ `assignee` - 负责人
- ✅ `agent` - 分配 Agent

**自动更新：**
- ✅ `updatedAt` - 最后更新时间
- ✅ `statusText` - 状态文本（根据 status 自动映射）
- ✅ `priorityText` - 优先级文本（根据 priority 自动映射）

## 📊 功能演示

### 1. 查看任务详情

**访问方式：**
1. 主页 → 任务列表 → 点击任务
2. 直接访问：`http://localhost:8080/task-detail.html?id=1`

**页面加载流程：**
```
1. 解析 URL 参数获取任务 ID
2. 调用 API /api/v1/tasks/{id}
3. 渲染任务信息
4. 加载评论列表
5. 加载附件列表
6. 加载任务历史
```

### 2. 更新任务状态

**操作：**
1. 右侧面板 → 状态操作
2. 点击目标状态按钮
3. 状态立即更新
4. 添加到历史记录

**数据流：**
```
用户点击 → Vue 更新本地状态 → 
调用 PUT API → 服务器保存 → 
更新成功提示
```

### 3. 发表评论

**操作：**
1. 评论区 → 输入内容
2. 点击"发表"按钮
3. 评论添加到列表顶部
4. 清空输入框

**数据流：**
```
用户输入 → 验证非空 → 
添加到本地列表 → 记录到历史 → 
（可选）调用 API 保存
```

### 4. 上传附件

**操作：**
1. 点击上传区域
2. 选择文件（支持多选）
3. 文件添加到列表
4. 显示文件名和大小

**支持：**
- ✅ 多文件上传
- ✅ 拖拽上传（待实现）
- ✅ 文件大小显示
- ✅ 文件类型图标
