### 14. Python 代码弃用警告 ✅ 已解决
- **问题：** 多个 Python 文件存在弃用警告
- **发现：**

  - `src/auth/auth.py:98,100,183` - datetime.utcnow() 已弃用

- **解决方案：**
  - 使用 `ConfigDict` 替代 `class Config`
  - 使用 `datetime.now(timezone.UTC)` 替代 `datetime.utcnow()`

- **状态：** 🟢 已完成

### 13. 页面显示为纯文本 ✅ 已解决
- **现象：** 浏览器显示 HTML 源代码而不是渲染页面
- **原因：** FastAPI 默认返回 JSON Content-Type
- **解决方案：** 使用 `FileResponse` 明确指定 `media_type="text/html; charset=utf-8"`
- **效果：** 页面正常渲染
